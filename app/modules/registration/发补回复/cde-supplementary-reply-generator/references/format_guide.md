# CDE发补回复模板格式规范详解

## 1. 页面设置

- **纸张大小**：A4 (210mm × 297mm)
- **页边距**：
  - 上：2.54cm (1英寸)
  - 下：2.54cm (1英寸)
  - 左：3.17cm (1.25英寸)
  - 右：3.17cm (1.25英寸)

## 2. 字体规范

### 2.1 字体类型
- **中文字体**：宋体 (SimSun)
- **西文字体**：Times New Roman

### 2.2 字号设置

| 元素 | 字号 | 磅值 |
|------|------|------|
| 封面大标题 | 二号 | 22pt |
| 封面副标题 | 三号 | 16pt |
| 封面信息 | 四号 | 14pt |
| 页眉文字 | 小四 | 12pt |
| 页脚页码 | 五号 | 10.5pt |
| 一级标题 | 小四 | 12pt |
| 二级标题 | 小四 | 12pt |
| 三级标题 | 小四 | 12pt |
| 正文内容 | 小四 | 12pt |

### 2.3 加粗设置
- 标题：全部加粗
- 【答复】【回复建议】标签：加粗
- 正文内容：不加粗

### 2.4 颜色设置
- **问题文本**：蓝色 (RGB: 0, 112, 192)
- **回复建议**：蓝色 (RGB: 0, 112, 192)
- **其他内容**：黑色

## 3. 段落格式

### 3.1 行间距
- 页眉：单倍行距
- 页脚：单倍行距
- 正文：1.5倍行距 或 固定值22磅

### 3.2 段前段后间距
- 所有段落：0磅

### 3.3 缩进设置
- 标题：无缩进
- 正文（回复/建议）：首行缩进2字符 (0.74cm)

### 3.4 对齐方式
- 页眉文字：左对齐
- 页眉页码：右对齐
- 页脚页码：居中
- 封面内容：居中
- 标题：左对齐
- 正文：两端对齐

## 4. 标题编号体系

### 4.1 一级标题
- **格式**：阿拉伯数字 + 顿号
- **示例**：1、证明性文件
- **字体**：宋体小四加粗

### 4.2 二级标题
- **格式**：括号 + 阿拉伯数字
- **示例**：（1）生产许可证
- **字体**：宋体小四加粗，蓝色

### 4.3 三级标题
- **格式**：圆圈数字
- **示例**：① 呋喃铵盐
- **字体**：宋体小四加粗，蓝色

## 5. 页眉格式

### 5.1 布局
- 左侧：公司logo
- 中间：三行文字（左对齐）
- 右侧：页码（第 X / Y 页）

### 5.2 文字内容
```
原料药登记资料
[药品名称]（无菌，原料药）
补充资料研究报告
```

### 5.3 格式
- 所有文字：宋体小四加粗
- 行间距：单倍
- 页眉下方：0.5倍行间距换行 + 横线

## 6. 封面格式

### 6.1 布局
内容垂直居中，包含：
1. 原料药登记资料（大标题）
2. 补充资料回复（副标题）
3. 产品信息（产品名称、登记号、受理号、申请人）
4. 公司名称

### 6.2 对齐
全部内容水平居中

## 7. 问题回复结构

每个问题的标准结构：

```
（1）[问题原文，蓝色加粗]

【答复】
[空白待填写，首行缩进2字符，两端对齐]

【回复建议】
• 回复思路：[分析CDE问题的核心关注点]
• 回复模板：[提供标准回复格式]
• 补充实验：[是/否]
• 实验项目：[如需补充，列出具体实验]
• 法规依据：[引用相关指南/药典]
```

## 8. 目录格式

### 8.1 标题
"目  录" - 宋体三号加粗，居中

### 8.2 内容
- 按标题层级生成
- 包含页码
- 可更新域

## 9. 常见问题检查

生成文档后检查：
- [ ] 所有问题均已收录
- [ ] 问题编号连续无跳号
- [ ] 问题文本为蓝色
- [ ] 页眉药品名称正确
- [ ] 页眉包含logo和页码
- [ ] 字体统一为宋体/Times New Roman
- [ ] 行间距一致
- [ ] 正文首行缩进正确
- [ ] 封面信息完整
